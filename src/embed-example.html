Embed keepthereceipts.org.za by pasting the following HTML code in an HTML embed block in your site:

<div id="keepthereceipts-embed"></div>
  <script type="text/javascript" src="https://pym.nprapps.org/pym.v1.min.js"></script>
  <script>
    var pymParent = new pym.Parent('keepthereceipts-embed', 'https://keepthereceipts.org.za/index.html', {});
  </script>
</div>
